# Glossary

> **Purpose**: Key terms, acronyms, and definitions used across AegisOS documentation.
> **Status**: ACTIVE · CANONICAL

---

**Navigation**: [Home](../Home.md) · **Reference** > Glossary

---

| Term | Definition |
|---|---|
| **AegisOS** | The AI Workstation Platform — enterprise-ready, local-first, privacy-preserving |
| **ADR** | Architecture Decision Record — a document capturing a significant architectural decision |
| **C2** | Command & Control — secure remote command execution subsystem |
| **Console** | The Next.js administrative dashboard (default port 3000) |
| **DPAPI** | Data Protection API — Windows machine-scope encryption |
| **HITL** | Human-in-the-Loop — approval gates requiring human authorization |
| **LiteLLM** | AI routing proxy that provides a unified OpenAI-compatible API across model providers |
| **MCP** | Model Context Protocol — standardized context server protocol |
| **Ollama** | Local AI inference engine for running GGUF models on GPU |
| **OX** | Operator Experience — the operational workflow and interface paradigm |
| **RBAC** | Role-Based Access Control — permission system based on user roles |
| **STRIDE** | Spoofing, Tampering, Repudiation, Information Disclosure, DoS, Elevation of Privilege |
| **Tailscale** | Mesh VPN for secure network connectivity between devices |
| **VRAM** | Video RAM — GPU memory used for model inference |
| **Zero Trust** | Security model where no request is trusted by default |
