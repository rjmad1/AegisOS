## Long-Term Vision & Research Themes
AegisOS is built to evolve from an autonomic single-workstation operating system into a distributed, multi-node enterprise knowledge work platform.

### Key Horizons (12–36 Months)
1. **Marketplace Cryptographic Package Signing**: End-to-end supply chain verification for 3rd-party agent extensions.
2. **Multi-Node Kubernetes Deployment Profile**: Horizontally scaling local inference across private cloud Kubernetes clusters.
3. **Autonomous Host Self-Refactoring Loops**: Verified, sandbox-tested local script refactoring loops with automated rollback protection.
4. **Formal Verification of Control Plane Safety**: Mathematical proofs verifying that execution paths in Layer 5 policy firewalls cannot be bypassed.
