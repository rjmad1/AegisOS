# AegisOS Knowledge Base: 01_PRODUCT_KNOWLEDGE.md

## Vision
AegisOS is the definitive enterprise-grade, local-first Autonomic AI Workstation Operating System — enabling organizations to run AI inference, agent orchestration, and system administration with full data sovereignty, zero-trust cryptographic security, and automated self-healing.

## Target Personas
1. **Software Engineers & DevOps Leads**: Require fast, context-aware code generation and CLI automation without leaking IP to public cloud LLMs.
2. **CISOs & Security Officers**: Require verifiable zero telemetry, HttpOnly JWT authentication, ECP prompt firewalls, and immutable audit trails.
3. **Platform Engineers & SREs**: Require low operational maintenance, zero configuration drift, and autonomic self-healing daemons.

## Key Features & Capability Matrix
* **7-Layer Autonomic Stack**: Hardware (L0) to Executive Plane (L6).
* **Executive Control Plane (ECP)**: Safety firewalls, rate limits, and output grounding scorecards.
* **Predictive VRAM Router**: Telemetry-based VRAM consumption velocity tracking and hybrid cloud bursting.
* **Autonomic Self-Healing**: Real-time service diagnostic sweeps and automatic fault recovery.
* **Enterprise SAML SSO**: Zero-touch group claim role mapping for Azure Entra ID / Okta.
* **Mobile C2 Cockpit**: Flutter mobile app with ECDSA command signing and biometrics.

## Product Evolution Horizons
* **Now (0-6m)**: Real-time hardware telemetry bus, zero-touch SAML group mapping, compliance auditor.
* **Next (6-18m)**: Visual canvas node editor, pre-warmed VM sandboxes, autonomic self-healing expansion.
* **Future (2-5y)**: Federated workstation compute mesh, intent-driven autonomous Saga engine.
