# AegisOS Studio Beta User Guide

Welcome to AegisOS Studio! This guide helps you navigate the AI Workspace environment.

---

## Getting Started

### 1. Dashboard Landing
When you open Studio, the **Workspace Intelligence Briefing** card gives you a quick snapshot of overnight events, index metrics, and running missions.

### 2. Guided Onboarding
If this is your first time loading the Studio, follow the onboarding tour guide. It will walk you through:
- Creating a workspace.
- Importing codebases and documents.
- Triggering knowledge indexing.
- Launching and watching a mission run.

### 3. Chronological Operational Timeline
Click **Activity** in the sidebar to review all system logs. 
- Use the **Search bar** to find specific steps.
- Use **Group By** to organize events by day, category, or mission.
- Click **Replay Mission** next to completed run events to review step-by-step telemetries.

### 4. Mission Replay Player
On the replay console, click **Play/Pause** to watch the mission execute. You can inspect:
- The exact prompt parsed (Intent Resolution).
- Agent thought logs and tool inputs/outputs.
- The cognitive scorecard (completeness, safety, and gaps).
