# Security Policy

## Supported Versions
We support security fixes for version 1.0.0 and above.

## Reporting a Vulnerability
Do not open a public Github issue to report a security vulnerability. Instead, email reports to [raja.jeevan@gmail.com] or contact the owner via [LinkedIn](https://www.linkedin.com/in/rajajeevankumar/).

Please include:
- A description of the issue.
- Step-by-step instructions to reproduce the vulnerability.
- Potential impact.

We will review your submission and respond within 48 hours.
